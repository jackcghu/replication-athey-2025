# Input Data

## Dataset: Oregon Health Insurance Experiment (OHIE)

The raw data file `1_Cleaned_wide_Dataset.csv` is included in this directory.

### Original Data Source
- **Paper:** Finkelstein, Amy, Katherine Baicker, Sarah Taubman, et al. 2018.
  Replication Data for: "Oregon Health Insurance Experiment."
- **URL:** https://doi.org/10.7910/DVN/SJG1ED
- **Repository:** Harvard Dataverse

### File Description
- `1_Cleaned_wide_Dataset.csv` — cleaned wide-format dataset with one row per individual
- N = 12,208 observations after exclusions (missing demographics, transgender individuals)
